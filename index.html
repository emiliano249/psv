Programa de Stock y Ventas

Aumenta la productividad de tu negocio con un sistema de gestión comercial fácil de usar, rápido y seguro.

Características:

Módulo de carga principal

Cuenta con una cómoda interfaz de carga de productos con ayuda automática de texto predictivo para ingresar datos relacionados de stock, evitando tipeo redundante. Funciones para cálculos de porcentajes sobre precios de los productos, descuentos, remarcación e IVA. Control de inventario de productos faltantes. Control de fecha de caducidad sobre el inventario de productos que se van a vender. Módulo de carga de Productos elaborados. Permite mediante un código de barra estándar o personalizado asociado a un nombre del producto vincular a varios productos para elaborar una promoción o artículos agrupados.

Módulo de ventas

Este módulo tiene una cómoda interfaz de venta para el operador o cajero detallando la suma de los productos ingresados y el vuelto que corresponde al cliente. Permite representar la cantidad de productos que están a la venta simplificando productos repetidos. Permite ventas de productos elaborados para promociones con códigos de barras personalizados. Permite buscar productos individuales por nombre o códigos de barra. Permite buscar productos agrupados por nombre o códigos de barra.

Módulo de reportes

Tiene funciones para ver reportes de ventas de diferentes secciones como artículos de mercado, sección carnicería, fiambrería, etc. Permite calcular el reporte de ventas entre diferentes fechas como: semanas, meses y años. Funciones de conexiones de la aplicación. Permite la conexión entre Operador o Cajero y servidor (Administrador general) a través de cable lan o wifi en un área de trabajo para carga de datos y ver reportes. Permite la conexión remota entre usuario y servidor (Administrador general) fuera del área de trabajo para carga de datos y ver reportes, lo cual permite trabajar desde su casa o a distancia.

Módulo de la interfaz de usuario

Cuenta con tres tipos de privilegios de usuario de acceso a la aplicación: Administrador General, Administrador (Encargado) y Operador (Cajero). Administrador General: tiene acceso a todas las configuraciones del sistema como establecer conexiones cajero y servidor, cambiar precios, ver reportes, configuración de driver fiscal epson y puede dar de alta a otros usuarios o eliminarlos. Administrador: pude hacer ventas, modificación de stock y ver reportes (no puede agregar otros usuarios ni establecer conexión cajero y servidor entre dos computadoras). Operador : Solo puede realizar ventas.

El programa permite hacer una copia de respaldo automática de toda la información en el disco local y en la nube.

Opciones adicionales

1) Aplicación para dispositivos android que se utiliza para verificar los precios de los productos sin la necesidad de estar en la caja o punto de venta.

2) Aplicación en telefono o tablet android para utilizarla como punto de venta, modificar precios, eliminarlos y ver reportes, puede trabajar desde casa.
